# 🎟️ Event Booking API

A RESTful API built using Node.js, Express, and MongoDB for managing events and bookings.

---

## 🚀 Features

- Create events
- View all events
- Book seats for events
- Seat availability validation

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
